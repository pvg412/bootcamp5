компоненты - новые, созданные нами объекты-теги. и уже в этих компонентах мы работаем.
компоненты бывают двух типов.
компоненты в контексте react - некоторые маленькие кирпичи, из которых состоит наше приложение
название компонентов в react пишутся с большой буквы
используя синтаксис в круглых скобках ( <div>
                                            hello!
                                        </div> ) мы можем писать в
